COMMENT AJOUTER VOS VRAIES PHOTOS
==================================

Déposez vos photos directement dans ce dossier "images/" en respectant
EXACTEMENT les noms ci-dessous. Dès qu'un fichier existe avec le bon nom,
il remplace automatiquement le bloc de couleur sur le site — vous n'avez
rien d'autre à modifier.

Format recommandé : .jpg, carré ou portrait, environ 1000x1000px, poids
inférieur à 300 Ko par photo (important pour la vitesse sur mobile en
Afrique de l'Ouest).

Pour ajouter un nouveau modèle ou une nouvelle couleur, il faut aussi
ajouter la liste des photos correspondante dans script.js (objet PRODUCTS),
dans le tableau "images: [...]" de la couleur concernée.
